# Locus Specificity (LOC)

**Locus Specificity (LOC)** is an Evidence Concept under
[Human Observational Data](index.md), covering how specifically the variant/locus
tracks with the phenotype and disease.

!!! note "Not yet modeled here"

    LOC is specified by the SVCv4 Standards but is **not yet covered by this data
    model**. This page summarizes it; detailed modeling is a later phase.

| Code | Captures |
|---|---|
| `LOC_PHE` | Observation of a specific phenotype. |
| `LOC_SEG` | Segregation of the variant with disease (co-segregation). |
